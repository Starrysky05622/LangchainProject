# LangChain Project

这是一个基于 LangChain 框架开发的项目。

## 项目简介

本项目利用 LangChain 框架构建应用程序，集成大语言模型（LLM）能力，实现问答、对话、文档处理等功能。

## 功能特性

- 基于 LangChain 的核心组件构建
- 支持多种大语言模型集成
- 提供链式调用能力
- 文档处理与向量存储支持
- 灵活的提示词模板

## 技术栈

- Python
- LangChain
- OpenAI API / 其他 LLM 提供商

## 安装

```bash
pip install langchain
pip install -r requirements.txt
```
